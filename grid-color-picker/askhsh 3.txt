Σε αυτή την άσκηση σας δινεται ενα αρχείο HTML με έτοιμο κώδικα HTML, CSS, και Javascript και θα πρέπει να τον συμπληρώσετε γραφοντας τον ορισμό 2 συναρτήσεων. Το αποτέλεσμα που θα βλεπει ο χρήστης είναι αυτό που φαινεται στην παρακάτω εικόνα:

1) Αν επιλεξουμε καποιο χρώμα μπορούμε να κανουμε κλικ σε καποια απο τα κελιά του πλέγματος και θα αλλάξει το background τους στο χρώμα που έχουμε επιλέξει. Αυτό μπορεί να γίνει όσες φορές θέλουμε.

2) Αν πατήσουμε το κουμπί που λέει Clear όλα τα τετραγωνα που εχουν χρωματιστεί αποκτούν το αρχικό χρώμα φόντου και έτσι χανουν τον "χρωματισμό" τους.



Υποδείξεις:

1) Όπως είπαμε με την "document.getElementById" μπορείτε να ανακτάτε το αντικείμενο που αντιστοιχεί σε ενα HTML element με την συγκεκριμένη τιμή για την attribute id

2) Με το property ".value" μπορείτε να παρετε το χρώμα που εχει επιλέξει ο χρήστης από το αντικείμενο που αντιστοιχεί στο συγκεκριμένο input που εχει type="color"

3) Απο το event object που δέχεται ως όρισμα ο event handler του κλικ σε ενα κελί έχετε πρόσβαση στο αντικείμενο του element του συγκεκριμένου κελιού

4) Θα προσθέσετε τον κώδικα που απαιτείται στο επισυναπτόμενο HTML αρχείο και θα το υποβαλετε στο eclass.
